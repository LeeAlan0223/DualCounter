# Dual Recursive Counter System
雙遞迴計數器系統


這是一個用 C 語言撰寫的雙遞迴計數器程式，具備靜態變數狀態儲存、輸入控制切換與模組化函式設計，適合作為中階程式學習與邏輯練習用途。

## ✨ 功能特色
- 使用者可自訂：
  - 總執行次數（c）
  - 每幾次交替切換計數器（newline）
  - 各計數器的起始值與遞增量
- 計數器間透過遞迴方式切換執行
- 靜態變數保留各自計數器的狀態
- 防呆判斷輸入是否為 1 / 2
- 自訂 pause 函式等待結束

## 🚀 使用方式

### 編譯：
```
gcc dual_counter.c -o dual_counter
```

### 執行：
```
./dual_counter
```

## 🧠 學習重點

- 靜態變數 static 在遞迴函式中的應用
- 全域變數與模組參數互動
- 函式指標與流程控制
- 程式流程圖可視化訓練邏輯力
